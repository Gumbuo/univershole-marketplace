# FoxHole's Pixel Characters and Maps Marketplace

A Next.js marketplace for selling premium pixel art game characters at $5 each.

## Features

- 16 animated character previews with cross-punch combat moves
- PayPal and Crypto (ETH) payment integration
- Cyan/green color scheme
- Commercial license included
- Responsive design

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- PayPal SDK
- Ethers.js
